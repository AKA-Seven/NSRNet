# NSRNet: Noise-Robust Stego-Image Restoration Network

## 📘 项目简介

**本科毕设题目**：噪声鲁棒的隐写域图像复原方法研究

本项目旨在构建一个具有噪声鲁棒性的隐写域图像修复网络 —— **NSRNet**，能够在隐写图像中存在高斯噪声干扰的情况下，在隐写域完成去噪、去模糊、超分辨率任务，并有效恢复秘密图像。
